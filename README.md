# DataDrivenQuadraticJSR.jl

## Workflow

### To run the GUI
```julia
(@v1.6) pkg> activate ./examples
```
Run the `main.jl` example by executing the corresponding file, or running
```julia
julia> include("./examples/main.jl")
```
